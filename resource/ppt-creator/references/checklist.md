# 课件自检清单（checklist）

> 出片前逐条过。P0 必须全过，否则不算完成。

## P0 — 必过（不过则课件不可用）

- [ ] **浏览器直接打开能翻页**：双击 html，方向键 ← → 正常，无 JS 报错
- [ ] **代码不溢出**：每页代码块无横向滚动条；长代码已分页
- [ ] **代码字号 ≥ 17px**：投影后排座能看清（template 默认 17px，别改小）
- [ ] **代码已着色**：关键字/类型/注释有区分，不是一坨灰字
- [ ] **关键行有高亮**：每段代码至少标出 1 处重点（`.hl`）
- [ ] **每页有 hud 页码**（封面、尾页除外），章节标签准确
- [ ] **算法有可视化**：讲算法的页必有图/表/步骤卡，不只靠文字
- [ ] **对比度达标**：浅底深字，深主题下文字 ≥ #e0e0e0；别用浅灰字配浅底

## P1 — 应过

- [ ] 每页一个要点，未堆砌
- [ ] 公式/复杂度用 `.formula`/`.big-o`，非纯文本凑合
- [ ] 易错点用 `.note bad`，正确做法用 `.note ok`，配对出现
- [ ] 封面有点题句、尾页有行动指引（不是干"谢谢"）
- [ ] SVG 节点图用 viewBox，填色用 accent 变量
- [ ] DP 表用 `table.grid`，答案格 `.fill`

## P2 — 建议

- [ ] 标题层级一致：封面 h1、内页 h2、卡片 h3
- [ ] 章节标签（hud .chap）反映真实分组，非全写"章节"
- [ ] 代码行的 `<` 用 `&lt;` 转义，未裸写
- [ ] 无残留占位文字（"在此替换""..."等）
- [ ] 文件名格式 `课件-<主题>-<日期>.html`

## P3 — 锦上添花

- [ ] 时间分配标注（教案场景）：hud 或卡片写 `⏱ 5min`
- [ ] 例题页有"为什么这样做"的注解，学生能自学
- [ ] 主题匹配场景（复习串讲用 T2 护眼等）
- [ ] 尾页 quote 有温度，呼应开场

## 常见翻车

| 现象 | 原因 | 修法 |
|------|------|------|
| 代码横向出滚动条 | 行太长 / pre 宽度撑爆 | 折行 `<br>` 或分页 |
| 翻页跳半页 | slide 不是 100vw | 检查 `flex:0 0 100%` 未被覆盖 |
| 代码没颜色 | 忘了加 span 类 | 套 `.kw`/`.ty`/`.cm` 等 |
| SVG 位置乱 | 没用 viewBox | 加 `viewBox="0 0 W H"` |
| 高亮行错位 | `.hl` 前后有缩进 | 该行顶格，不留空格 |
| 字太小 | 改了默认字号 | 恢复 template 默认值 |
