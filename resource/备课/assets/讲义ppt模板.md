## 三、类的使用(30分钟)


### (一)、开胃菜

```cpp
class 宠物
{
public:
    string 名字;
    int 年龄;    
    宠物(string 取的名字, int 多大了)
    {
        名字 = 取的名字;
        年龄 = 多大了;
    }    
    void 自我介绍()
    { 
        cout << "我是" << 名字 << "，今年" << 年龄 << "岁啦！" << endl;
    }    
    void 卖萌()
    {
        cout << 名字 << "：主人主人，快来陪我玩嘛~" << endl;
    }
};
```



```cpp
// 宠物类
int main()
{
    宠物 小猫咪("咪咪", 2);
    小猫咪.自我介绍();
    小猫咪.卖萌();
    return 0;
}
```

> 📌 当我们需要创建多个具有相同属性和行为的对象时，可以使用类来定义模板。
> 例如在游戏中创建多个宠物角色，每个宠物都有名字和年龄属性，以及自我介绍和卖萌等行为。这种方式比为每个宠物单独定义变量和函数更加高效和便于管理。


---

**总结**：

- 类模板和类的语法相似，在声明模板template后面加类，此类称为类模板
- 类模板使用只能用显示指定类型方式
- 类模板中的模板参数列表可以有默认参数

> 📌 使用场景说明：类模板适用于需要为多种数据类型提供相同功能的场景。例如STL中的容器类(vector、list、map等)，它们可以存储不同类型的数据(int、string、自定义类型等)但操作方式相同。通过类模板，我们可以编写一次代码就能支持多种数据类型，大大提高了代码的复用性和灵活性。在开发通用数据结构和算法库时非常有用。
