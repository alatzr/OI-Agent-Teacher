# 知识卡片模板

> 每张卡片一个知识点。用学生自己的理解角度写，不是抄教材。
> 文件名：`<知识点>.md`，如 `前缀和.md`、`单调栈.md`。

```markdown
# <知识点>

## 一句话
_<能让学生 5 秒想起来的定义>_

## 学生的理解角度
- _当时是怎么让他听懂的？用了什么类比/例子？_

## 易错点 / 坑
- _（从 error_log 或对话中提炼）_

## 关联
- 题目：<题号/题名>
- 错题指针：见 `error_log.md` 的 <日期> 条目
- 相似知识点：<>

## 复习调度
- 上次接触：YYYY-MM-DD
- 掌握状态：❌/📖/⚠️/✅/💪/🏆
- 下次复习：YYYY-MM-DD
- 复习历史：第 1 次 YYYY-MM-DD / 第 2 次 ...
```

## 写卡要点

1. **定义用学生的嘴**：不是"前缀和是对前缀的累加"，而是"把前面所有数加起来存着，要区间和就相减"
2. **易错点写具体**：不是"注意边界"，而是"下标从 1 开始算，差分会漏掉第一个元素"
3. **关联留指针**：能从卡片跳到错题、跳到题号，别让它孤立
4. **复习状态跟着 knowledge_matrix 走**：两处状态保持一致
